# gnNews

News aggregator

## Demo

https://gnnews-sz.vercel.app

## Run Locally

Clone the project

```bash
  git clone https://github.com/szymonkwak/gnNews.git
```

Go to the project directory

```bash
  cd gnNews
```

Install dependencies

```bash
  npm install
```

Start the development server

```bash
  npm run dev
```

Run tests

```bash
  npm run test
```
